# JavaScript Fundamentals

Este repositório reúne exercícios e pequenos desafios em JavaScript, com foco em reforçar os fundamentos da linguagem e a lógica de programação.

## Objetivo
- Relembrar e consolidar conceitos fundamentais de JavaScript
- Praticar resolução de problemas e algoritmos
- Manter histórico de aprendizado versionado com Git

## Conteúdo
- Exercícios do Exercism
- Desafios do freeCodeCamp
- Scripts simples para prática de lógica e linguagem

## Tecnologias
- JavaScript (ES6+)
- Git / GitHub

## Observações
Este é um repositório de estudo contínuo. Os códigos podem evoluir conforme o aprofundamento nos conceitos.
